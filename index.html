jPlusUI 介绍

组件化的开发能有效提升开发效率，降低维护成本。

轻量

组件底层代码为 34K（Gzipped）, 其它平均每个组件的大小为 1-5 k。

稳定

组件底层拥有完整的单元测试，确保各个浏览器的兼容问题。并且每次升级都能保证向下兼容。

易上手

本着拿来即用的原则，每个组件只需复制示例代码即可见到效果。三分钟快速上手

易定制

所有组件均按需加载。代码遵循 HTML5 规范，组织构架清晰，容易扩展和定制。

1.0 版本新特性

  • 增加大量示例和 API 说明。
  • 重新全面测试浏览器的兼容问题。
  • 全面简化底层构架，减少继承层次。
  • 全面简化 less 引用层次，删除大部分的 @import 。
  • 改进开发和演示界面，并使用 xfly 框架重写服务器。
