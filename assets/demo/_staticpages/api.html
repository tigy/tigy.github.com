API 的文字说明

用法

assert(bValue, msg);

参数

bValue

(Object) 值。肯能的值：

  • AAA: (Object) 配置对象。
注意：必须输入。
msg

(String) 错误后的提示。默认为"断言失败"。

返回值

(Boolean) 返回 bValue 。

默认值

false

所有配置

名称 说明 继承
rootPath

System 安装的根目录, 可以为相对目录。
stackTrace

是否在 assert 失败时显示函数调用堆栈。
useLess

是否使用 lesscss

所有字段

名称 说明 继承
scripts

全部已载入的名字空间。
styles

全部已载入的样式。

示例

assert(true, "{value} 错误。", value);
