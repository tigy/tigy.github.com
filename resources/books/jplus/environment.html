浏览器

J+ 支持所有主流浏览器，但在最新版浏览器能获得最佳体验。

最低支持版本

  • Internet Explorer 6+
  • Firefox 3.6+ (PC, Mac)
  • Chrome 10+
  • Safari 5+
  • Opera 10.5+ (PC, Mac)

建议开发者使用

  • FireFox 4 + FireBug (推荐)
  • Chrome 10+
  • Internet Explorer 9+
  • Safari 5+

服务器

J+ 的正常运行不需要服务器坏境。但是我们强烈建议在服务器坏境下调试开发，因为J+开发版会自动载入被引入的代码模块，而在file:///协议下，这个功能将不能使用。服务器可以是任意的HTTP服务器，如 Apache 。

为了方便开发， 组件管理系统自带了一个简单的 web 服务器。

  • Windows 用户: 双击执行 assets/bin/startserver.bat 即可打开服务器。
  • Linux 用户: 打开 assets/bin-linux/startserver.sh 即可打开服务器。
  • Mac 用户: 打开 assets/bin-mac/startserver 即可打开服务器。
