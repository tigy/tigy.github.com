jQuery 中的 API 在 J+ 中的等效替代。
jQuery J+
jQuery(selector/nodelist) Dom.query(selector/nodelist)
jQuery(html) Dom.parse(html)
jQuery(function) Dom.ready(function)
is(expr) match(expr)
after(content) insert('afterEnd', content)
before(content) insert('beforeBegin', content)
prepend(content) insert('afterBegin', content)
children(expr) children(children)
filter(filter) filter(filter)
next(expr) next(expr)
parent(expr) parent(expr)
prev(expr) prev(expr)
siblings(expr) siblings(expr)
