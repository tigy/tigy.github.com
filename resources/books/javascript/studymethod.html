学 Javascript 主要是学什么

由于其它原因，很多初学者认为学 Javascript 就是用来在网页上做效果的。 其实这是片面的理解。 Javascript 只是一个语言，虽然当初它是被设计用在网页交互的，但现在，它已经是 一个完整的计算机脚本语言。Javascript不只用于网页上。

作为一门语言，必须要学会它的基础语法。此外，因为 Javascript 最主要用途是 DOM 操作 和 Ajax ， 所以，有必要额外学习 DOM 和 Ajax 的知识。不然即使学会一门语言，也没有用武之地。

学习 Javascript ,所需要掌握的知识有:

  • 初级: 掌握基本的 Javascript 语法。
  • 中级: 掌握 DOM 操作 和 Ajax 操作。
  • 高级: 掌握 Javascript 语法中的一些特性和难点。
  • 骨灰级: 精通各浏览器对Javascript的兼容问题，以及常见的 Javascript 技巧。

初级部分: 语法

初级掌握基本的 Javascript 语法。要求会用 Javascript 写一些简单的逻辑代码，比如计算1+2+3+...+100。初级部分主要是看书。最好是去书店买一本，实在不愿掏钱可以从网上下一个 pdf 版本。至于什么书，这里也不额外推荐， 自己能看懂的书就是好书。也不要指望看完哪个书就直接进入高级，知识总是慢慢积累的。

中级部分: DOM 和 AJAX

中级部分主要还是看书和源码。读一段优秀的代码可以受益匪浅。当然还需要自己实践。
