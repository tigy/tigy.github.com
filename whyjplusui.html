  • Q：我只需要一个浮层、日期选择器这样一些简单的组件。

    A：网上有很多现成组件。但是：

    1. 有些组件功能多，代码大，而我只需要最简单的功能。
    2. 组件的风格不是我想要的。
    3. 组件之间风格不统一，而且也有很多重复的代码。
    4. 个人写的组件用到我的项目里就会出各种问题，最后还是得自己写。

  • Q：我希望在通用组件基础上继续做一些和业务有关的组件。

    A：虽然每个公司都有自己的组件库，但都是根据自身业务需要而做的，特别是很多代码都加上了公司的 logo。而像 Extjs, jQuery UI 这样的知名 UI 框架，用它现成的组件很方便，但在其基础上添加自己的组件却很麻烦。

    做组件更主要是要有一套完整的组件体系：从组件的开发到使用的完整解决方案。
